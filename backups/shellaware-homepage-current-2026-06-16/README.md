# ShellAware Turtle Journey Homepage

An immersive React homepage for ShellAware, an SDG 14 sea turtle conservation awareness campaign.

## Run

```bash
npm install
npm run dev
```

## Notes

This version uses online image URLs, so it runs with internet without needing local images. Replace image URLs later with files inside `src/assets/images/` if you want offline images.

## Included

- Homepage only
- Hero section
- Turtle journey intro
- Threat education
- Public action tips
- Expert highlights
- Shelly chatbot preview only
- Campaign call to action
- Footer

## Not included

- Login/signup
- Dashboard
- Accounts
- Real chatbot functionality
- Backend/database
